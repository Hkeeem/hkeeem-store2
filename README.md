# Hkeeem Store
متجر جاهز Next.js + Supabase

## الرفع على GitHub
1. فك الضغط
2. ارفع كل الملفات على github.com/Hkeeem/Hkeeem عبر uploading an existing file
3. ارجع Vercel واضغط Deploy

## متغيرات البيئة المطلوبة في Vercel
NEXT_PUBLIC_SUPABASE_URL
NEXT_PUBLIC_SUPABASE_ANON_KEY
SUPABASE_SERVICE_ROLE_KEY
OPENAI_API_KEY=temp_disabled
WHATSAPP_TOKEN=temp_disabled
WHATSAPP_PHONE_ID=temp_disabled
